# DataViz Toolkit

> A comprehensive data visualization library for Python

[![Documentation Status](https://readthedocs.org/projects/dataviz/badge/?version=latest)](https://dataviz.readthedocs.io/)
[![PyPI version](https://badge.fury.io/py/dataviz.svg)](https://badge.fury.io/py/dataviz)

## Overview

DataViz Toolkit makes it easy to create beautiful, interactive visualizations with just a few lines of code.

## Installation

```bash
pip install dataviz-toolkit
```

## Quick Start

```python
import dataviz as dv

# Create a simple bar chart
chart = dv.BarChart(data, x='category', y='value')
chart.show()
```
